# 검색 결과 확인 방법

## 1. 애플리케이션 로그에서 확인할 로그

다음 로그들을 찾아서 확인하세요:

### 검색 쿼리 확인
```
Google Search API 검색 쿼리: ...
```

### API 응답 확인
```
Google Search API 응답 길이: ... bytes
Google Search API 응답 JSON 구조 확인:
  - root 필드들: ...
  - items 노드 존재: ..., 타입: ...
```

### 검색 결과 확인
```
검색 결과 items 배열 크기: ...
```
또는
```
검색 결과에 items가 없거나 비어있습니다.
응답 전체 구조 (처음 1000자): ...
```

## 2. 검색 결과가 0개인 가능한 원인

1. **Google Custom Search API가 결과를 반환하지 않음**
   - 검색 쿼리가 너무 구체적이거나 특이할 수 있음
   - API 키나 Search Engine ID 문제

2. **API 응답 구조가 예상과 다름**
   - `items` 필드가 없거나 다른 이름일 수 있음
   - 에러 응답일 수 있음

3. **검색 쿼리 문제**
   - AI가 추천한 제품명이 검색에 적합하지 않을 수 있음

## 3. 해결 방법

로그에서 다음을 확인하세요:
- "Google Search API 검색 쿼리" 로그에서 실제 검색 쿼리 확인
- "root 필드들" 로그에서 응답 구조 확인
- "items 노드 존재" 로그에서 items 배열 존재 여부 확인
